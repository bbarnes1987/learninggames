Timeline Game Builder

Fill in the settings and card details, then generate a ready-to-copy version of the timeline game.

1. Game settings

These are the main settings that change the look and wording of the game.

This appears on every card and in the game instructions.
Example: 5 means every incorrect drop adds 5 seconds to the final score.

2. Card details

Choose how many cards you want, then enter the year, event text and image link or upload an image for each card. If two cards have the same year, they will be ordered in the same order they appear below.

For each card, teachers can paste a Google Drive share link or upload an image. Uploaded images are embedded into the generated HTML. Blank images use a scroll emoji placeholder.

3. Generate and copy the game code

Click generate, then copy the full HTML code into your website embed area, or download it as an HTML file.
