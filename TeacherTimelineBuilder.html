2. Card details

Choose how many cards you want, then enter the label to show on the card, event text and image link or upload an image for each card. The correct answer is based on the order of the cards in this builder: Card 1 comes first, Card 2 comes second, and so on.

For each card, teachers can paste a Google Drive share link or upload an image. Uploaded images are embedded into the generated HTML. Blank images use a scroll emoji placeholder.

3. Generate and copy the game code

Click generate, then copy the full HTML code into your website embed area, or download it as an HTML file.
